采用 Sanic 搭建的一个轻量论坛 ~~（只有一部分的后端）~~ 。

## 安装

### 环境与依赖项

项目采用 Sanic 框架编写，同时部分内容也需要以下的库：

* SQLAlchemy, ORM
* alembic, 数据库迁移
* Pydantic, 接口的检查
* pyyaml, 加载位于项目根目录的 `instance.yaml` 作为实例的配置
* python-jose, 令牌的生成
